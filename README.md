# AI-exploratory

Repositório de estudos exploratórios sobre datasets públicos de Inteligência Artificial.

## Objetivo

Explorar, analisar e visualizar datasets públicos relacionados ao impacto da IA na economia, mercado de trabalho, e sociedade — com foco especial em perspectivas para o Brasil e América Latina.

## Estudos

| Pasta | Dataset | Descrição |
|-------|---------|-----------|
| [01_anthropic_economic_index](./01_anthropic_economic_index/) | Anthropic/EconomicIndex | Análise do uso de IA por ocupação e setor econômico, com foco no Brasil vs. países comparáveis |

## Como usar

```bash
# Instalar dependências
pip install -r requirements.txt

# Iniciar Jupyter
jupyter notebook
```

## Estrutura geral

```
AI-exploratory/
├── requirements.txt
├── 01_anthropic_economic_index/

└── ...
```
